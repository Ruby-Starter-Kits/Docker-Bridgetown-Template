## Setup & Running Locally

Clone down the repo, install [Docker](https://hub.docker.com/editions/community/docker-ce-desktop-mac/) & run:

```bash
docker-compose up
```

Then navigate your browser to https://127.0.0.1:4000/ to see your site.
